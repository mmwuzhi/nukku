# Changelog

## 0.1.0

### English

- Initial public release of Nukku for macOS 26 Tahoe.
- Adds the fixed-canvas notch panel, hover or click expansion, and full-screen compatible window behavior.
- Adds media controls backed by MediaRemote, browser MediaSession support, artwork caching, and safe app-level fallback copy.
- Adds File Drop, Calendar, Camera, HUD overlays, Launch at Login, preferences, and localized app strings for English, Japanese, and Simplified Chinese.
- Adds GitHub Actions builds and tagged release packaging for `Nukku.app.zip`.

### 日本語

- macOS 26 Tahoe向けNukkuの最初の公開リリースです。
- 固定キャンバスのノッチパネル、ホバーまたはクリックでの展開、フルスクリーン対応のウィンドウ動作を追加しました。
- MediaRemoteベースのメディア操作、ブラウザMediaSession対応、アートワークキャッシュ、安全なアプリ単位のフォールバック表示を追加しました。
- File Drop、カレンダー、カメラ、HUD表示、ログイン時に起動、環境設定、英語、日本語、簡体字中国語のアプリ内文言を追加しました。
- GitHub Actionsによるビルドと、tag付きリリースでの `Nukku.app.zip` パッケージ作成を追加しました。

### 简体中文

- Nukku 面向 macOS 26 Tahoe 的第一个公开版本。
- 加入固定画布刘海面板、鼠标悬停或点击展开，以及兼容全屏空间的窗口行为。
- 加入基于 MediaRemote 的媒体控制、浏览器 MediaSession 支持、封面缓存，以及安全的 app 级兜底文案。
- 加入 File Drop、日历、摄像头、HUD 覆盖层、登录时启动、偏好设置，以及英文、日文、简体中文 app 文案。
- 加入 GitHub Actions 构建，以及通过 tag 发布时生成 `Nukku.app.zip` 的流程。
